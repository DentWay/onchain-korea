## 4. 🟡 기술 개발 & 최적화

### 4.1 번들 사이즈 최적화 (550KB → 420KB)

**액션 (우선순위순):**

1. **Code splitting — 라우트별 lazy loading** (4/17~18)
   - `React.lazy()` + `Suspense` 적용
   - `/admin/`, `/dashboard/`, `/certificate/` 청크 분리
   - 초기 로드 200KB 감소 목표

2. **의존성 감사** (4/18)
   - `npm audit` + `npx depcheck`
   - 미사용 라이브러리 제거
   - lodash → native JS 대체

3. **이미지 최적화** (4/19)
   - 로고/아이콘 SVG 스프라이트 생성
   - 썸네일 AVIF 변환
   - 브랜드 에셋 (이미 리프레시됨) 최적화 확인

**검증:** `npm run build` → bundle analyzer 리포트

### 4.2 모바일 반응형 개선 (4/19~4/20)

**현황:** 데스크톱 우수 ✅ / 모바일 82% ⚠️

**LCP 개선 (목표: <3.5s):**
- Hero 이미지 WebP 변환 + lazy loading
- 폰트 preload (Pretendard CDN 캐시)
- Supabase API 타임아웃 최적화 (4s → 2s)

**CLS 개선 (목표: <0.05):**
- 스켈레톤 로딩 UI 추가
- 동적 높이 요소 aspect-ratio CSS 명시

**모바일 UI:**
- 레슨 카드: 2열 → 1열 (터치 영역 확대)
- 버튼: 최소 48×48px 확보
- 수평 스크롤 제거 (iPhone SE 기준)

**검증:** Lighthouse 모바일 90+ 달성
