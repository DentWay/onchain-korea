# Onchain Korea — 블록체인, 처음부터 안전하게

Blockchain Literacy & Safety Education Platform for Korea.
Built by Elixi Venture Studio Group × Greed Academy.

## Quick Start

```bash
npm install
npm run dev
```

Then open `http://localhost:5173`

## Pages

| Route | Page | Description |
|---|---|---|
| `/` | Dashboard | 진행률, 4주 커리큘럼, 히든 토픽 |
| `/week/:id` | Week Detail | 주차별 학습 콘텐츠 + 실습 목록 |
| `/action/:id` | Action Guide | 스텝바이스텝 실습 가이드 |
| `/hidden` | Hidden Topics | 매주 핫이슈 아티클 + 포럼 |
| `/community` | Community | 카카오톡, 뉴스레터, 포럼, 스터디 |
| `/certificate` | Certificate | 온체인 수료증 미리보기 |

## Stack

- React 18 + Vite 5
- Tailwind CSS 3
- React Router 6
- Lucide React (icons)
- Pretendard (Korean font)

## Curriculum (Greed Academy based)

| Week | Content | Action |
|---|---|---|
| 1 | Wallet Hygiene + Wallets (S2-L1, S1-L5) | Phantom setup, burner wallet |
| 2 | Onchain Explorers + DYOR (S2-L6, S1-L7/8) | TX reading, scam ID |
| 3 | DeFi + Digital Assets + NFTs (S2-L4, S2-L2, S1-L6) | DEX swap, NFT mint |
| 4 | Stablecoins + Cross-Chain + Staking (S2-L7, S2-L8, S1-L1) | Bridge, staking |
