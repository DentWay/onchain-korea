# 07_법무컴플라이언스 (Legal & Compliance)

## 부서 개요

온체인 코리아의 법적 준수, 규제 대응, 계약 관리를 담당하는 부서입니다.
- 한국 암호화폐 규제 동향 모니터링
- Proof of Attendance (PoA) 법적 지위 정의
- 이용약관 및 개인정보보호 정책
- 그랜트 계약 및 파트너십 법무 검토
- 콘텐츠 출처 관리 및 저작권 준수

## 현재 상태

### 완료된 작업
- ✅ Proof of Attendance 용어 정의 ("공식 자격증 아님" 명시)
- ✅ 그랜트 계약 체결 ($3,017)
- ✅ Greed Academy 협력 계약
- ✅ 한국 암호화폐 규제 기초 문서화
  - VAUPA (2024.7.19 시행)
  - 디지털자산기본법 (협의 중)
  - 암호화폐 과세 (2027년 시행 예정)

### 진행 중인 작업

- 🔄 파트너십 계약 법무 검토 (거래소, DeFi)

### 다음 우선순위

## 핵심 규제 사항

### 1. VAUPA (가상자산이용자보호법) - 2024.7.19 시행
- 거래소: 보안 감시, 사용자 자산 분리
- 지갑 서비스: 개인 관리 거래소와 유사 규제
- 온체인 교육 플랫폼: 직접 적용 X (거래 없음)
- **하지만**: cNFT Proof of Attendance는 "토큰" 발행이므로 주의 필요

### 2. 디지털자산기본법 (협의 중)
- 스테이블코인 규제권 협의 (금융감독청 vs 기획재정부)
- 예상 시행: 2026년~2027년
- 현재: 영향 분석 중

### 3. 암호화폐 과세 (2027년 시행 예정)
- 수익 과세: 양도소득세 또는 소득세 (확정 대기)
- 교육 플랫폼: 간접 영향 (학습자가 세금 책임)
- 수강료: 비과세 예상 (교육 서비스)

## 주의사항

1. **Proof of Attendance 명시**
   - "이는 공식 자격증이 아닙니다"를 모든 고지에 명시
   - NFT 발행과 규제의 관계 모니터링

2. **콘텐츠 출처 관리**
   - Greed Academy 29개 소스는 모두 명시 출처
   - 각 레슨에 "원문 읽기" 링크 필수

3. **거래소 파트너십**
   - Upbit/Bithumb과 제휴시 VAUPA 준수 필수
   - 계약서에 규제 변화 조항 추가

4. **cNFT 민팅**
   - Solana NFT는 한국에서 "토큰" 카테고리 (규제 대상)
   - 학습 증명용 NFT는 "실제 금융 가치 없음" 명시 필요
