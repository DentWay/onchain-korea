# OnChain Korea Brand Notes

These notes capture the usable rules from the current brand PDF and screenshots.

## Naming
- Product name: `OnChain Korea`
- Landing-only co-brand lockup: `OnChain Korea` with `by Elixi.`
- App shell branding: `OnChain Korea` only

## Visual Rules
- Keep the mark red/blue only: `#CD313A` and `#0047A0`
- Keep the product ink dark: `#0C0D11`
- Use low-hierarchy text at roughly 38% opacity on light and dark surfaces
- Preserve the symbolic language: network nodes, chain links, cube references
- Do not add gradients, glow, rotation, skew, or logo effects
- In the live lockup, the Taegeuk symbol must sit optically as the first glyph of `nChain`; do not use padded assets or wide flex gaps between the symbol and the `n`.
- The symbol should sit slightly above the `n` baseline so it reads as an `O`, not as a detached badge.

## Theme Split
- Landing: light, editorial, brand-led
- Learning app: dark, denser utility UI
- `by Elixi.` appears only in landing lockups and landing certificate preview

## Landing Hero Invariants
- Do not modify or cover the existing 3D interactive feature when revising the landing page.
- Do not place the hero brand lockup, hero headline, or overlay panels inside the active 3D interference zone.
- Keep the hero brand block visually below the main 3D object so the interaction remains readable and unobstructed.
