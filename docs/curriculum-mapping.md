## 매핑이 이렇게 된 이유

### S1-L5 (Wallets) → w1-1, w1-2
하나의 "Wallets" 아티클에서 (1) 지갑 개념과 (2) 클라이언트 비교를 분리. 초보자에게는 "지갑이 뭔지"와 "어떤 앱을 쓸지"가 별개의 질문이라 2개로 나눔.

### S2-L1 (Wallet Hygiene) → w1-4, w1-5
"Hygiene" 아티클에서 (1) 버너 지갑이라는 구체적 실습과 (2) 전반적 보안 습관을 분리. 버너 지갑은 practice 타입으로 별도 실습 가능.

### S2-L4 (DeFi) → w3-1, w3-2
DeFi 개념(AMM, LP)과 실제 스왑 실습을 분리. 이론을 먼저 배우고 바로 실습하는 흐름.

### S1-L6 (NFTs) → w3-4, w3-5
NFT 개념 이해와 Metaplex Core 민팅을 분리. 이해 → 실습 순서.

### S2-L6 (Onchain Explorers) → w2-1, w2-2
블록 탐색기 사용법과 고급 온체인 분석을 분리. 기초(트랜잭션 읽기) → 심화(고래 추적, 토큰 흐름).
