**Current Status:** MVP 운영 중 (무료 교육)  
**Target Cohort:** 3기 수강생 모집 중 (목표 200명)  
**Prepared by:** Elixi Venture Studio Group × Greed Academy

- **비용 구조:** [추정]

## 2. 잠재 수익 모델 분석

### 2.1 Model 1: Freemium (기본 무료 + 프리미엄 유료)

**개념:** 기초 8주 과정 무료 → 고급 과정/자격증 유료

#### 구조

| 레벨 | 내용 | 가격 | 고객 | 예상 전환율 |
|------|------|------|------|----------|
| **Basic** | 8주 기초 커리큘럼 | 무료 | 신규 사용자 | - |
| **Pro** | 심화 4주 (프로젝트 분석) | ₩99,000/월 | 관심 학생 | 15~20% [추정] |
| **Enterprise** | 1:1 멘토링 + 포트폴리오 | ₩499,000/월 | 실전 투자자 | 5~10% [추정] |
| **Certification** | 수료증 + 자격증 (1회) | ₩149,000 | 기초 완료 | 30~40% [추정] |

#### 재무 시뮬레이션 (Year 1)

**가정:**
- 기초 과정 총 수강자: 600명
- Pro 전환율: 18% [추정] = 108명
- Pro 평균 구독 기간: 3개월
- Certification 전환율: 35% [추정] = 210명

**수익:**
- Pro 구독: 108명 × ₩99,000 × 3개월 = **KRW 3.2억**
- Certification: 210명 × ₩149,000 = **KRW 3.1억**
- **Year 1 추정 수익: KRW 6.3억**

**평가:**
- ✓ 신규 수익 창출
- ✓ 사용자 기반 활용
- ✗ 강사 품질 관리 필요 (비용 증가)
- ✗ 프리미엄 콘텐츠 제작 노력 필요

### 2.2 Model 2: B2B 거래소 스폰서십 & 파트너십

**개념:** Upbit, Coinone 등 거래소가 "우리 사용자를 위한 안전 교육" 프로그램 스폰서

#### 구조

| 파트너 | 형태 | 가격 | 내용 |
|--------|------|------|------|
| **거래소** | 교과과정 스폰서 | KRW 5,000만~1억 | 자사 사용자 대상 기초 과정 |
| **금융회사** | HR 교육 프로그램 | KRW 2,000만~5,000만 | 직원 대상 블록체인 리터러시 |
| **정부 기관** | 시민 교육 위탁 | KRW 1억~3억 | FSC/금융위원회 공동 프로젝트 |

#### 거래소 스폰서십 상세

**Upbit 파트너십 예시:**
- 목표: "Upbit 신규 가입자 안전 교육 프로그램"
- 제안: OnChain Korea 8주 과정 + Upbit 보안 실습 세션
- 수수료 모델:
  - **평판료 (Base Fee):** KRW 5,000만 (분기)
  - **성과 연동:** 완료자당 ₩50,000 × 완료 수강자
  - **예상:** 분기 200명 × ₩50,000 = KRW 1억 (+ 기본료)

**Year 1 추정 (2개 거래소 파트너십):**
- Upbit: KRW 5,000만 × 4분기 + 성과비 KRW 2억 = **KRW 4억**
- Coinone: KRW 3,000만 × 4분기 + 성과비 KRW 1억 = **KRW 2.2억**
- **Year 1 B2B 수익: KRW 6.2억**

**평가:**
- ✓ 고정 수익 안정화
- ✓ 사용자 자동 획득 (거래소 지렛대)
- ✓ 거래소와 정부 규제 정렬
- ✗ 거래소 협상 시간 길어질 수 있음
- ✗ 거래소 의존도 증가 (리스크)

### 2.3 Model 3: 자격증/자격명 프로그램 (Certification)

**개념:** "OnChain Korea Blockchain Safety Certification" 공식 자격증

#### 구조

**Tier 1: Basic Certificate (기초)**
- 수강 기간: 8주
- 가격: ₩149,000 (일회)
- 내용: 수료증 + 디지털 배지
- 예상 대상: 기초 완료자 35% [추정]

**Tier 2: Professional Certificate (전문가)**
- 수강 기간: 추가 4주 심화
- 가격: ₩299,000 (일회)
- 내용: 전자 자격증 + 직무 추천 네트워크
- 예상 대상: Pro 수강자의 50% [추정]

**Tier 3: Master Certificate (마스터)**
- 수강 기간: 추가 8주 고급
- 가격: ₩499,000 (일회)
- 내용: 물리 자격증 + 1:1 멘토링 + 커뮤니티 VIP
- 예상 대상: Enterprise 수강자의 40% [추정]

#### 인정 체계 구축

**국내 인정:**
- FSC(금융감독원) 협력 → "금융회사 직원 교육" 인정 [추정]
- 거래소 공식 인정 자격증 (Upbit, Coinone)

**국제 인정:**
- 블록체인 교육 연합 (IBTEC 같은 국제 기구) 인정 신청 [추정]

#### Year 1 수익 추정

- Basic Cert: 210명 × ₩149,000 = **KRW 3.1억**
- Professional Cert: 54명 × ₩299,000 = **KRW 1.6억**
- Master Cert: 10명 × ₩499,000 = **KRW 0.5억**
- **Year 1 Certification 수익: KRW 5.2억**

**평가:**
- ✓ 반복적 수익 스트림 (각 코호트마다)
- ✓ 직업 경쟁력 향상 → 사용자 만족도 ↑
- ✗ 자격증 공신력 구축에 시간 필요 (1~2년)
- ✗ 거래소/정부 협력 필수

## 3. 경쟁사 벤치마킹: 수익 모델 분석

### 3.1 국내 거래소 교육 서비스

**Upbit Care Centre (Upbit)**
- **모델:** 무료 교육 + 고객 서비스
- **수익화:** 거래 수수료에 포함 (교육은 마케팅 비용)
- **특징:** 자사 거래소 편향, 깊이 제한적

**Coinone Academy (Coinone)**
- **모델:** 무료 강좌 + 정보 제공
- **수익화:** 직접 교육 수익 아님
- **특징:** 보안 중심, 정보성

**평가:** 기존 거래소 교육은 거래 유도 목적 → OnChain Korea의 중립성이 차별화

### 3.2 해외 블록체인 교육 플랫폼

**Coursera - Blockchain Specialization**
- **모델:** Freemium (기초 무료, 수료증 유료)
- **가격:** ₩180,000~300,000 (3개월)
- **수익:** 강사 수익분배 + 플랫폼 수수료 (25~30%)

**CoinMarketCap Academy (Binance)**
- **모델:** 무료 강좌 + NFT 배지 + 광고/거래소 전환
- **수익화:** 인접 거래소로의 트래픽 유도

**Ledger Academy (Ledger)**
- **모델:** 무료 교육 콘텐츠 + 제품 판매
- **수익화:** 교육을 통한 제품(하드웨어 지갑) 판매 유도

**평가:**
- 해외: Freemium (수료증 유료) 모델이 주류
- 가격대: ₩150,000~500,000 범위
- 참고: OnChain Korea도 비슷한 가격대 적용 가능

## 4. 추천 단계별 수익화 전략 (Phased Approach)

### Phase 1: 사용자 기반 확보 (Q2~Q3 2026)

**예상 수익:** KRW 0 (VC 자금에 의존)

**투자:** KRW 1.5억 (기존 운영비)
