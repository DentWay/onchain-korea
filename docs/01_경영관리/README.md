# 01_경영관리 (Business Management)

## 부서 개요

온체인 코리아 프로젝트의 전략, 의사결정, 로드맵을 담당하는 경영관리 부서입니다.
- 3단계 마스터플랜 (Phase 1~3) 수립 및 실행
- 아키텍처 및 전략 의사결정 기록
- 분기별 마일스톤 추적
- 외부 파트너십 및 커리큘럼 소스 관리

## 현재 상태

### 완료된 작업
- ✅ Phase 1 (4주 → 8주 커리큘럼 확장) 운영 중
- ✅ GA 29개 소스를 내부 라이브러리로 통합
- ✅ 12가지 주요 전략적 결정 기록 (의사결정 문서)
- ✅ 8주 커리큘럼 매핑 완료 (순서 재구성 포함)

### 진행 중인 작업
- 🔄 Phase 2 콘텐츠 작성 (6월~8월 2026)
  - 24개 새로운 레슨 (금융 기초 → RWA/STO)
  - 16개 액션 가이드
  - ~200개 퀴즈 문항 및 중간/기말 고사
- 🔄 Phase 1 운영 최적화 (전환율, 완료율)

### 다음 우선순위
1. Phase 2 DeFi 심화 커리큘럼 최종 확정 (4월)
2. 파트너십 협의 (한국 거래소, DeFi 프로토콜) - Phase 3 준비
3. 월별 성과 추적 시스템 수립
