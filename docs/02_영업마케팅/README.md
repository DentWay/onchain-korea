# 02_영업마케팅 (Sales & Marketing)

## 부서 개요

온체인 코리아의 랜딩페이지 전환율 최적화, 콘텐츠 마케팅, 수강생 모집을 담당하는 부서입니다.
- 랜딩페이지 전환율 전략 (Cialdini 심리학 원칙 적용)
- 콘텐츠 팩트체크 및 신뢰도 관리
- 마케팅 캠페인 운영 (Instagram, X, KakaoTalk)
- 한국 시장 맞춤 메시징 및 UX 카피
- 구매 퍼널 최적화
